# usePopcorn

A movie search app. Search films using the OMDB API, rate them with a star rating, and save your watched list to localStorage.

Built while learning React — based on a course project.

## What I practiced
- Data fetching with useEffect
- Custom hooks (useMovies, useLocalStorageState, useKey)
- Reusable star rating component
- localStorage for persistence
- Cleanup functions and AbortController

## Tech
- React 18
- OMDB API
- Create React App

## Run locally

```bash
npm install
npm start
```
